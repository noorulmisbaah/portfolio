Restaurant Website

The Restaurant Website, named Noor Kitchen, is a website where snacks, desserts, and food can be purchased. Visitors can search for items, can create an account or sign in and can also make comments or view comments posted by others. Once signed in, the user will be taken to the shopping area where purchasable items are displayed. Items information can be viewed. The user will add an item to the shopping cart if wished to purchase. A transaction receipt will be downloaded immediately if the purchase is successful. There is an icon where the users can be taken to their dashboard. Users can view their transactions history or delete any record they don't need. Users can also update or delete their account.

Timer Application

The Timer Application, named Noor Timer, is a web timer application where users can create a timers. It looks very similar to a stop watch. But unlike a stop watch Noor Timer allows users to create multiples timers at the same time. Each timer will have a title and a description. The application is built with a customized menu. Users can update or delete their account if they wish.

School Website

The School Website is built with basic functionalities that most school websites have. It is built with a complex Students Results Management System (SRMS). This is where students' results are processed, uploaded, updated, downloaded, and deleted. The website is also with a section where the school financial records are managed. This is where the financial officer will enter a financial record, fetch a record, and view all the school financial records.
